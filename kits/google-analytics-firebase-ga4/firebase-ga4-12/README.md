# mParticle-FirebaseGA4 (firebase-ga4-12)

This kit integrates [Firebase Analytics (GA4)](https://firebase.google.com/docs/analytics) with the [mParticle Apple SDK](https://github.com/mParticle/mparticle-apple-sdk). It targets Firebase SDK major version 12.x.

## Adding the Kit

Add the `mParticle-FirebaseGA4` product from this package to your Xcode project or `Package.swift`:

```swift
.package(url: "https://github.com/mParticle/mparticle-apple-integration-google-analytics-firebase-ga4-12", .upToNextMajor(from: "9.0.0"))
```
